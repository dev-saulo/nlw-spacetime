## 🖥️ Projeto

Esse é um projeto Web responsivo de uma capsula do tempo para exibir memórias em uma timeline

## 🚀 Tecnologias

Esse projeto foi desenvolvido durante a NLW da Rocketseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
